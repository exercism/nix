# Learning

## Recommended Resources

- [nix.dev](https://nix.dev) — official tutorials and guides for learning Nix
- [Nix Pills](https://nixos.org/guides/nix-pills/) — a progressive deep-dive introduction to Nix
- [Nix Reference Manual](https://nix.dev/manual/nix/stable/language/) — complete language reference
- [Learn X in Y Minutes: Nix](https://learnxinyminutes.com/docs/nix/) — quick syntax overview
- [Nixpkgs Library Functions](https://nixos.org/manual/nixpkgs/stable/#part-functions) — `lib` functions used in tests
