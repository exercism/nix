# Instructions

Reverse a string.

For example, the string "robot" becomes "tobor".

Implement the `reverse` function that takes a string and returns the string with characters in reversed order.

Note: Nix has no built-in string reversal function, so you will need to implement it yourself using builtins such as `builtins.stringLength`, `builtins.substring`, `builtins.genList`, and `builtins.concatStringsSep`.
