## Getting Help

If you'd like help solving the exercise, check the following resources:

- The [Nix language reference](https://nix.dev/manual/nix/stable/language/index.html)
- [Nixpkgs Library Functions](https://nixos.org/manual/nixpkgs/stable/#part-functions)
- The [Exercism community forum](https://forum.exercism.org) for language-specific questions

## Submitting Incomplete Solutions

It's possible to submit an incomplete solution so you can see how others have completed the exercise.
